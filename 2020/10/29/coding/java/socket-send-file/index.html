Java Socket 传输文件

描述

基于Java Socket传输文件/文件夹，包括客户端、服务端，具备以下特点。

  • 一次连接，传输多文件，在传输大量小文件场景下，可节约大量连接创建时间及资源消耗
  • 支持传输文件夹，但会跳过空文件夹
  • 服务端支持多线程
  • 支持结果回显

实现

依赖

程序集成在Maven工程内，lombok非必需，仅用于提升代码整洁度，如剔除需少量改动代码。

基础信息

用于传送每次请求附带参数。

服务端

服务端包含FileTransferServer.java 和 FileTransferServerThread.java 两个文件，用于实现多线程支持。
